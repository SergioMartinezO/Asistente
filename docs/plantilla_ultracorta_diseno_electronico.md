# Plantilla ultra-corta — Diseño electrónico (uso rápido)

Usa este formato cuando necesites pedir un diseño completo en muy pocas líneas:

1. Proyecto: [NOMBRE]
2. Objetivo funcional: [QUÉ DEBE HACER]
3. Entorno/usuario: [DÓNDE Y PARA QUIÉN]
4. Inicio del proyecto: [YYYY-MM-DD]
5. Restricciones: [energía, costo, conectividad, seguridad]
6. Exige plan por fases con fecha inicio/fin (Hardware, Software, Diagramas, Word, Web)
7. Exige entregables verificables por fase
8. Exige componentes con justificación técnica
9. Exige código + diagramas + Word + Web
10. Cierre obligatorio: "Proyecto completo y listo para ejecución"
